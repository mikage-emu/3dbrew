# Request

| Index Word | Description                |
|------------|----------------------------|
| 0          | Header code \[0x04090000\] |

# Response

| Index Word | Description     |
|------------|-----------------|
| 0          | Header code     |
| 1          | Result code     |
| 2          | The network CRC |

# Description

Gets the loaded network's CRC.
