# Request

| Index Word | Description                |
|------------|----------------------------|
| 0          | Header code \[0x005600C0\] |
| 1          | u64, Title ID              |
| 2          | u16, Content Index         |

# Response

| Index Word | Description              |
|------------|--------------------------|
| 0          | Header code              |
| 1          | Result code              |
| 2          | bool, Has Content Rights |
