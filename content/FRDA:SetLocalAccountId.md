# Request

| Index Word | Description                |
|------------|----------------------------|
| 0          | Header code \[0x04030040\] |
| 1          | Local account ID           |

# Response

| Index Word | Description |
|------------|-------------|
| 0          | Header code |
| 1          | Result code |

# Description

This switches the current local account being used on friends to the
given one.
