# Request

| Index Word | Description                |
|------------|----------------------------|
| 0          | Header code \[0x00020000\] |

# Response

| Index Word | Description                                             |
|------------|---------------------------------------------------------|
| 0          | Header code                                             |
| 1          | Result code                                             |
| 2          | Online state (8-bit, 0 = not online, non-zero = online) |
