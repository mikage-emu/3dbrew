# Request

| Index Word | Description                |
|------------|----------------------------|
| 0          | Header code \[0x08410000\] |

# Response

| Index Word | Description |
|------------|-------------|
| 0          | Header code |
| 1          | Result code |

# Description

This deletes the "/Nintendo 3DS/<SomeID>/" directory from SD card.
