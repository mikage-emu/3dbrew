# Request

| Index Word | Description                |
|------------|----------------------------|
| 0          | Header code \[0x08120080\] |
| 1-2        | u64, Archive Handle        |

# Response

| Index Word | Description     |
|------------|-----------------|
| 0          | Header code     |
| 1          | Result code     |
| 2-3        | u64, Free Bytes |
