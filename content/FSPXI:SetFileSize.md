# Request

| Index Word | Description                |
|------------|----------------------------|
| 0          | Header code \[0x000E0100\] |
| 1          | Size lower word            |
| 2          | Size upper word            |
| 3          | File handle lower word     |
| 4          | File handle upper word     |

# Response

| Index Word | Description |
|------------|-------------|
| 0          | Header code |
| 1          | Result code |
