# Request

| Index Word | Description                        |
|------------|------------------------------------|
| 0          | Header code \[0x000D0040\]         |
| 1          | Screen index (top = 1, bottom = 2) |

# Response

| Index Word | Description |
|------------|-------------|
| 0          | Header code |
| 1          | Result code |

# Description

Restore LCD configuration from GSP memory.
