# Request

| Index Word | Description                            |
|------------|----------------------------------------|
| 0          | Header code \[0x000400C0\]             |
| 1          | Screen (1 = top, 2 = bottom, 3 = both) |
| 2          | Min                                    |
| 3          | Max                                    |

# Response

| Index Word | Description |
|------------|-------------|
| 0          | Header code |
| 1          | Result code |
