# Request

| Index Word | Description                                            |
|------------|--------------------------------------------------------|
| 0          | Header code \[0x00080042\]                             |
| 1          | HTTP context handle                                    |
| 2          | 0x20, processID translate-header for the ARM11-kernel. |
| 3          | processID set by the ARM11-kernel.                     |
