# Request

| Index Word | Description                |
|------------|----------------------------|
| 0          | Header code \[0x000F0000\] |

# Response

| Index Word | Description                   |
|------------|-------------------------------|
| 0          | Header code                   |
| 1          | Result code                   |
| 2          | Current process disable count |
| 3          | Total disable count           |
