# Request

| Index Word | Description                |
|------------|----------------------------|
| 0          | Header code \[0x04050000\] |

# Response

| Index Word | Description                |
|------------|----------------------------|
| 0          | Header code \[0x04050040\] |
| 1          | Result code                |

# Description

This deletes the amiibo registration info, including related data like
the application area.
