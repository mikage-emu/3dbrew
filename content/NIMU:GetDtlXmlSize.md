# Request

| Index Word | Description                |
|------------|----------------------------|
| 0          | Header code \[0x00230000\] |

# Response

| Index Word | Description                |
|------------|----------------------------|
| 0          | Header code \[0x002300C0\] |
| 1          | Result code                |
| 2-3        | XML Size                   |

# Description

Gets the size of the auto title delivery XML saved in dtl.xml.
