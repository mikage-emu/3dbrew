# Request

| Index Word | Description                |
|------------|----------------------------|
| 0          | Header code \[0x000C0000\] |

# Response

| Index Word | Description |
|------------|-------------|
| 0          | Header code |
| 1          | Result code |
| 2          | Step count  |

# Description

This returns the total steps taken with this system, this value is
displayed in the Activity Log application as "Steps Taken".
