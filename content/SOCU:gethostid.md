# Request

| Index Word | Description                |
|------------|----------------------------|
| 0          | Header code \[0x00160000\] |

# Response

| Index Word | Description                                       |
|------------|---------------------------------------------------|
| 0          | Header code                                       |
| 1          | Result code                                       |
| 2          | POSIX return value (32-bit output LAN IP address) |
